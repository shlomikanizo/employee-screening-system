## 🚀 התקנה מהירה (3 דקות!)

### אופציה 1: Gmail (הכי פופולרי)

#### שלב 1: צור App Password ב-Gmail

1. **עבור ל-Google Account**
   ```
   https://myaccount.google.com/
   ```

2. **Security** (אבטחה) → **2-Step Verification** (אימות דו-שלבי)
   - אם עדיין לא הפעלת, **הפעל עכשיו**

3. **App passwords** (סיסמאות אפליקציות)
   - בחר: **Mail** (דואר)
   - בחר: **Windows Computer** (מחשב Windows)
   - לחץ **Generate** (צור)

4. **שמור את הסיסמה בת 16 התווים!**
   ```
   לדוגמה: abcd efgh ijkl mnop
   ```

#### שלב 2: עדכן את backend/.env

ערוך `backend\.env`:
```env
# אפשר שימוש באימייל
USE_EMAIL=true

# הגדרות Gmail
SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
EMAIL_USER=your-email@gmail.com
EMAIL_PASSWORD=abcd efgh ijkl mnop
EMAIL_FROM_NAME=מערכת סינון מועמדים
```

**החלף:**
- `your-email@gmail.com` - עם האימייל שלך
- `abcd efgh ijkl mnop` - עם ה-App Password שקיבלת

#### שלב 3: זהו!

הרץ את Backend:
```powershell
start_backend.bat
```

המערכת תשלח הודעות באימייל! 🎉

## 🆘 פתרון בעיות

### אימייל לא נשלח

#### בעיה 1: "Authentication failed"
```
❌ Email authentication failed - check credentials
```

**פתרון:**
1. ✅ וודא שה-App Password נכון (16 תווים, ללא רווחים)
2. ✅ וודא שה-Email נכון
3. ✅ נסה ליצור App Password חדש

#### בעיה 2: "Email service is disabled"
```
⚠️ Email service enabled but EMAIL_USER not configured
```

**פתרון:**
1. ✅ וודא ש-`USE_EMAIL=true` ב-.env
2. ✅ וודא ש-`EMAIL_USER` מוגדר
3. ✅ הפעל מחדש את Backend

#### בעיה 3: אימייל הגיע ל-Spam

**פתרון:**
1. סמן כ-"Not Spam" (לא דואר זבל)
2. הוסף את השולח לאנשי קשר
3. בפעם הבאה יגיע לתיבה הרגילה

#### בעיה 4: Google חוסם

**פתרון:**
1. אפשר "Less secure app access" (אפליקציות פחות מאובטחות)
2. או עבור ל-App Password (מומלץ!)

## 🔄 שילוב עם WhatsApp

### אפשר לשלב את שני השירותים!

```env
# שלח באימייל כברירת מחדל
USE_EMAIL=true
EMAIL_USER=your-email@gmail.com
EMAIL_PASSWORD=your-app-password

# אבל גם רשום את WhatsApp (כגיבוי)
USE_WHATSAPP_WEB=false
WHATSAPP_ACCESS_TOKEN=your-token
WHATSAPP_PHONE_NUMBER_ID=your-id
```

המערכת תשתמש באימייל, אבל אפשר לשנות ל-WhatsApp בכל רגע!
