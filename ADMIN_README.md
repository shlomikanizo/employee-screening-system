# 🎛️ מערכת ניהול דינאמית (Admin Dashboard)

## סקירה כללית

מערכת ניהול מקצועית ומלאה לניהול טפסים, מועמדים, וסטטיסטיקות בזמן אמת.

---

## 🚀 גישה למערכת הניהול

### דרך הדפדפן
```
http://localhost:3000/admin
```

---

## ✨ תכונות עיקריות

### 📊 Dashboard - סקירה כללית
- **סטטיסטיקות בזמן אמת**
  - סה"כ טפסים (פעילים/לא פעילים)
  - סה"כ שליחות
  - שיעור הצלחה בסינון
  - שיעור אישור תנאים
  
- **גרפים מתקדמים**
  - Progress bars אינטראקטיביים
  - התפלגות נתונים
  - מגמות והשוואות

- **טפסים ושליחות אחרונות**
  - רשימת טפסים לאחרונה
  - טבלת שליחות אחרונות
  - פילטור וחיפוש

### 📝 ניהול טפסים
#### יצירת טופס חדש
1. לחץ על "טופס חדש" בפינה הימנית
2. מלא את כל השדות:
   - מידע בסיסי על המשרה
   - פרטי חברה
   - דרישות מקדמיות
   - שאלות סינון
   - תנאי עבודה והטבות
3. לחץ "שמור"

#### עריכת טופס קיים
1. בחר טופס מהרשימה
2. לחץ "ערוך"
3. ערוך את השדות הרצויים
4. לחץ "שמור"

#### אפשרויות נוספות
- **תצוגה מקדימה** - פתיחת הטופס בחלון חדש
- **העתקת קישור** - העתקה מהירה של קישור הטופס
- **מחיקה** - מחיקה רכה (סימון כלא פעיל)

### 👥 ניהול שליחות
- **טבלה מפורטת** עם כל השליחות
- **פילטרים**:
  - הכל
  - עברו סינון
  - נכשלו בסינון
  - אישרו תנאים
  
- **פרטים מלאים**:
  - שם, טלפון, אימייל, עיר
  - סטטוס סינון ותנאים
  - תאריך שליחה
  - תשובות לשאלות

- **צפייה בפרטים**:
  - לחץ על עין 👁️ לצפייה מלאה
  - Modal עם כל התשובות

### ⚙️ הגדרות
- עיצוב מותאם אישית (בקרוב)
- הגדרות מערכת (בקרוב)
- ניהול משתמשים (בקרוב)

```javascript
// מבנה טופס חדש
{
  "job_title": "מפתח Full Stack",

  "job_location": "תל אביב",

  "company_name": "שם החברה",
  "company_address": "כתובת החברה",
  "company_logo_url": "https://...",
  "screening_questions": [
    {
      "id": "q1",
      "question": "האם יש לך ניסיון?",
      "type": "boolean",
      "required": true,
      "correct_answer": true
    }
  ],
  "requirements": [
    {
      "text": "דרישה 1",
      "is_mandatory": true

  ],
  "terms_and_conditions": {

  }

```

### סוגי שאלות זמינים

#### 1. כן/לא (Boolean)
```json
{
  "type": "boolean",
  "question": "האם יש לך ניסיון?",
  "correct_answer": true
}
```

#### 2. בחירה (Select)
```json
{
  "type": "select",
  "question": "באילו טכנולוגיות?",
  "options": ["Python", "JavaScript", "Java"]
}
```

#### 3. טקסט חופשי (Text)
```json
{
  "type": "text",
  "question": "ספר על עצמך"
}
```

#### 4. מספר (Number)
```json
{
  "type": "number",
  "question": "כמה שנות ניסיון?"
}
```

## 🔧 API Endpoints

### Forms Management
```http
GET    /api/admin/forms              # קבלת כל הטפסים
GET    /api/admin/forms?is_active=true  # טפסים פעילים בלבד
PUT    /api/admin/forms/{id}         # עדכון טופס
DELETE /api/admin/forms/{id}         # מחיקת טופס (רכה)
```

### Submissions
```http
GET    /api/admin/submissions        # קבלת כל השליחות
GET    /api/admin/submissions?passed_screening=true  # שעברו סינון
GET    /api/admin/forms/{id}/submissions  # שליחות לטופס מסוים
GET    /api/admin/submissions/{id}   # שליחה ספציפית
```

### Statistics
```http
GET    /api/admin/stats/overview     # סטטיסטיקות כלליות
GET    /api/admin/stats/forms/{id}   # סטטיסטיקות לטופס
```

## 🐛 פתרון בעיות

### Admin Dashboard לא נטען
1. וודא ש-Backend רץ על port 8000
2. בדוק שהנתיב הוא `/admin` בדיוק
3. פתח Console ובדוק שגיאות
4. רענן דף (Ctrl+F5)

### לא רואה טפסים/שליחות
1. וודא שיש טפסים ב-Database
2. בדוק שה-API מחזיר נתונים: `http://localhost:8000/api/admin/forms`
3. בדוק Network tab בDevTools

### שגיאה בשמירת טופס
1. וודא שכל השדות החובה מלאים
2. בדוק את הפורמט של `screening_questions` ו-`requirements`
3. בדוק שה-JSON תקין
